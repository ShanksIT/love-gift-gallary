# 07. Acceptance Checklist

Dùng checklist này để kiểm tra sau khi Codex generate code.

## Functional Checklist

- [ ] Mở trực tiếp `index.html` bằng browser được.
- [ ] Opening screen hiển thị đầu tiên.
- [ ] Button “Mở món quà của anh” hoạt động.
- [ ] Main content hiện sau khi click.
- [ ] Nhạc nền phát sau khi click nếu file tồn tại.
- [ ] Có nút bật/tắt nhạc.
- [ ] Love Counter tính đúng theo `startDate`.
- [ ] Timeline render từ config.
- [ ] Gallery render từ config.
- [ ] Click ảnh mở modal/lightbox.
- [ ] Reasons render từ config.
- [ ] Letter render từ config.
- [ ] Final button mở popup.
- [ ] Floating hearts hoặc hiệu ứng romantic hoạt động.

## UI Checklist

- [ ] Giao diện romantic, mềm mại.
- [ ] Màu sắc thống nhất.
- [ ] Font dễ đọc.
- [ ] Card có spacing tốt.
- [ ] Không bị rối animation.
- [ ] Hero section tạo cảm giác bất ngờ.
- [ ] Letter section giống một bức thư tình.
- [ ] Gallery ảnh không bị méo.

## Responsive Checklist

- [ ] Mobile width 375px hiển thị tốt.
- [ ] Tablet width 768px hiển thị tốt.
- [ ] Desktop width 1440px hiển thị tốt.
- [ ] Button dễ bấm trên mobile.
- [ ] Timeline không bị vỡ layout.
- [ ] Gallery không tràn màn hình.

## Robustness Checklist

- [ ] Nếu thiếu file nhạc, website vẫn chạy.
- [ ] Nếu ảnh bị lỗi, layout không vỡ.
- [ ] Không có lỗi JavaScript nghiêm trọng trong console.
- [ ] Không phụ thuộc internet nếu không dùng external font/library.
- [ ] Nếu dùng CDN, có fallback hoặc ghi chú trong README.

## Content Checklist

- [ ] Đã thay tên người nhận.
- [ ] Đã thay ngày bắt đầu yêu.
- [ ] Đã thay timeline thật.
- [ ] Đã thay caption ảnh thật.
- [ ] Đã thay thư tình thật.
- [ ] Đã thay câu hỏi/lời hứa cuối.
